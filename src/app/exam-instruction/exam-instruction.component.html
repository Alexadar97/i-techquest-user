Instructions

    1. There will be no negative marking

    2. There will be “Start exam”option to start the exam

    3. Question types: as of now a.Single choice (options will have radio button to select) b.Multiple choice (options will have check box to select the answers)

    4. Once Start your exam you can’t pause your exam, time will be running automatically

    5. There will be a timer running at the upper-right corner of the exam screen

    6. You can go to random questions in between the exam by typing question number in the top

    7. Flag for Review option available for all the questions. If you want to review the particular question in the future then you can click the Flag for review button. To Unflag click the button again

    8. “End Exam” option is in last question page. You can navigate to the previous questions using Previous button option and same for going next.

    9. There will be a submit feedback option available for all the questions. User can submit feedback for each and every questions
